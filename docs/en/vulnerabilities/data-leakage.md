# Data Leakage

## Overview

Attacks and incidents where sensitive information (API keys, personal data, conversation history, etc.) is unintentionally leaked externally.

## Risk in OpenClaw

- System prompt/configuration leakage through LLM
- Sensitive information output to log files
- Improper storage of backup data
- Information leakage in agent-to-agent communication
