# Vulnerability Cases

Collecting and analyzing vulnerability cases related to LLM applications and chatbots.

## Categories

- [Prompt Injection](prompt-injection.md) - Attacks that rewrite LLM instructions
- [Sandbox Escape](sandbox-escape.md) - Attacks that break container isolation
- [Dependency Vulnerabilities](dependency.md) - Vulnerabilities in npm packages, etc.
- [Authentication Bypass](auth-bypass.md) - Circumventing authentication/authorization
- [Data Leakage](data-leakage.md) - Unintended leakage of sensitive information
