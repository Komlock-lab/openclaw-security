# サンドボックスエスケープ

## 概要

サンドボックスエスケープは、コンテナやサンドボックス環境から脱出し、ホストシステムにアクセスする攻撃。
OpenClawはDockerベースのサンドボックスを提供しているが、設定や実装の不備により突破される可能性がある。

## OpenClawにおけるリスク

- `sandbox.mode: off`（デフォルト）ではサンドボックスなしで実行
- Dockerソケットのマウントによるコンテナエスケープ
- ボリュームマウント経由のホストファイルアクセス
- 特権コンテナの悪用
