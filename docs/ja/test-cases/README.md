# テストケース

OpenClawに対するセキュリティ検証テストケースの一覧です。各ガイドにはステップごとの手順とPASS/FAILの判定基準があり、誰でも自分のインスタンスをテストできます。

## カテゴリ別テストケース

| カテゴリ | テスト数 | 状態 | ガイド |
|---------|---------|------|-------|
| プロンプトインジェクション | 27 | 完成 | [テストガイド](prompt-injection-guide.md) |
| サンドボックスエスケープ | - | 準備中 | - |
| スキルシステム悪用 | - | 準備中 | - |
| チャンネルなりすまし | - | 準備中 | - |
| シークレット漏洩 | - | 準備中 | - |
| Agent通信乗っ取り | - | 準備中 | - |

## はじめ方

セキュリティテストが初めてですか？ まずは**プロンプトインジェクション テストガイド**から始めてみてください。初級から上級まで27テストを収録し、所要時間は約60〜90分、特別なツールは不要です。

1. [AuditClawスキャン](../skill/README.md)を実行して現在のリスクレベルを把握する
2. [プロンプトインジェクション テストガイド](prompt-injection-guide.md)に沿ってテストを実施する
3. 結果を記録し、失敗したテストの対策手順を適用する
