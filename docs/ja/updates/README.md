# アップデート情報

OpenClawおよび関連ライブラリのセキュリティアップデート情報を追跡しています。

## バージョン脆弱性チェッカー

自分が使っているOpenClawのバージョンにどんな脆弱性があるか、すぐに確認できます。

```bash
# スクリプトで確認
npx tsx scripts/version-check.ts 2026.2.10

# Claude Codeコマンドで確認
/check-version 2026.2.10
```

## バージョン履歴の概要

OpenClawは2025年11月に誕生し、名前を4回変えています。

| 期間 | 名前 | バージョン形式 |
|------|------|---------------|
| 2025-11 | warelay | セマンティック (v0.1.1~) |
| 2025-12 | clawdis | セマンティック (v2.0.0-beta~) |
| 2026-01 | clawdbot | CalVer (2026.1.5~) |
| 2026-01-29~ | **openclaw** | CalVer (2026.1.29~) |

現在はほぼ毎日リリースされています。
