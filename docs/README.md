# OpenClaw Security Knowledge Base

OpenClawのセキュリティに関するナレッジベースです。

## このドキュメントについて

このナレッジベースでは、以下の情報を体系的に整理・公開しています：

- **脆弱性事例**: LLMアプリケーション・チャットボットに対する攻撃手法と対策
- **アップデート情報**: OpenClawおよび関連ライブラリのセキュリティアップデート
- **ベストプラクティス**: OWASP、NIST等のセキュリティガイドラインに基づく推奨事項
- **テストケース**: セキュリティ検証のための攻撃テストケースと結果

## 対象プロジェクト

[OpenClaw](https://github.com/openclaw/openclaw) - オープンソースのパーソナルAIアシスタント

## 攻撃カテゴリ

| カテゴリ | 説明 |
|---------|------|
| プロンプトインジェクション | LLMの指示を書き換える攻撃 |
| サンドボックスエスケープ | コンテナ分離を突破する攻撃 |
| スキルシステム悪用 | スキル機能を悪用した攻撃 |
| チャンネルなりすまし | メッセージングチャンネルの偽装 |
| シークレット漏洩 | APIキー等の機密情報の窃取 |
| Agent通信乗っ取り | Agent-to-Agent通信の傍受・改ざん |
